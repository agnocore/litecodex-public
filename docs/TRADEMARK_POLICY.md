# Lite Codex Trademark Policy

Status: Frozen  
Effective date: 2026-04-22

## 1. Protected Marks
- `Lite Codex`
- `litecodex-entry`
- 官方 Logo、官方发布标识、官方签名通道标识

## 2. Allowed Use
- 可以在事实性描述中使用 “compatible with Lite Codex CE”。
- 可以在文档中说明基于 Lite Codex CE 修改，但必须明确“非官方发行”。

## 3. Prohibited Use
- 禁止 fork 后继续使用 `Lite Codex` 作为产品主名称进行发布。
- 禁止在域名、安装器、更新源、产品页中冒充“官方 Lite Codex”。
- 禁止使用官方 Logo、官方签名标识、官方通道名称误导用户来源。

## 4. Fork Naming Rule
- Fork/衍生版必须采用独立名称与独立标识。
- 必须在首页、CLI `--version`、更新提示中标注 “Unofficial build / Community fork”。

## 5. Distribution Rule
- 二次分发可包含源码与可执行文件，但不得附带官方商标授权暗示。
- 任何“官方认证”“官方更新源”“官方支持”字样，需有发行方书面授权。

## 6. Violation Handling
- 侵权发布将触发下架与商标维权流程。
- 商标违规可构成 CE 发布阻断项。
